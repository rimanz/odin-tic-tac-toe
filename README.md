# Tic Tac Toe

A simple web-based Tic Tac Toe game built with HTML, CSS, and JavaScript as an assignment project for [The Odin Project](https://www.theodinproject.com/) curriculum.

**Live Demo:** https://rimanz.github.io/odin-tic-tac-toe

## Features

- Two-player game (X and O)
- Optional player name input via setup modal
- Interactive 3x3 grid with hover effects
- Displays active player name or symbol
- Detects and announces winner
- Draw detection
- Modal dialogs for game setup and results
- New Game button to restart with a new setup
- Reset Game button to clear the board without changing player names

## How to Play

1. Open [index.html](index.html) in a web browser.
2. (Optional) Enter player names in the setup modal and click "Start Game", or click "Start Game" to use default X and O symbols.
3. Click on any empty tile to place your mark.
4. The game alternates between players. The active player is displayed at the top.
5. First to get three in a row (horizontally, vertically, or diagonally) wins.
6. If all tiles are filled without a winner, it's a draw.
7. When the game ends, a result modal displays the winner or draw message.
8. Click "Play Again" to start a new game with another setup modal, or "Reset Game" to restart with the same players.
